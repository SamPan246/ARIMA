Learn _________

Machine learning serves many purposes, one specific role being that it predicts and forecasts data. An algorithm known as ARIMA, which stands for AutoRegressive Integrated Moving Average, is specialized for time series forecasting. Time series data is essentially data points in a time sequence which ARIMA can analyze and forecast based on four variables; p, d, q, and s.

P

The autoregressive component of ARIMA is represented in the order variable known as p, which indicates how many past data points in the time series are actually used in the model to make a forecast. In other words, the value of p determines how far back the model goes to analyze past data, which aids it in predictions. Another forecast predictor includes forecast errors, which are the differences between a predicted value and the actual one.

Q

The moving average in ARIMA is specified through the variable of q, which determines the number of past forecast errors(or fluctuations in data) in the time series that are used to predict. For instance, in a stock perspective, a higher q value would indicate that there are more spikes/dips in the data, meaning the model would rely more on them to make better predictions.

D

The d variable resembles the integrated average, which is a metric of how many times the time series data is made stationary. This process, also known as differencing, involves subtracting a previous data value from the current one, making the values have constant means and variance, making the overall time series easy to analyze and predict via ARIMA. Essentially, stationary data means that the dependent variable has no relationship with time(IV), and stays constant.

S

Finally, the s variable in ARIMA accounts for the seasonal order of the data, which measures the number of time units in a particular seasonal cycle. For instance, if data was analyzed in yearly seasonality, the s variable would be 12 for monthly data.

Example:

With the use of these 4 variables, ARIMA is able to analyze data and make future predictions upon it. A primary source of time-series data is stock prices, which ARIMA can forecast the difference between its predictions and actual values of them. Let’s walk through an example, analyzing a model through a stock lens. A standard ARIMA model is 1,0,0, following the order of p, d, and q. In this case, p being set to one means that the model only looks as far back as one previous day to predict the current stock price(essentially uses yesterday to predict today). D in this model is set as 0, meaning there is no differencing involved with the data, and it is already stationary. Lastly, q is also set to 0 here, meaning that it doesn’t rely at all on past forecast errors in predicting stock prices. You might wonder, where is the fourth variable, S? In order to add seasonality to the model, you would need to configure another set of three seasonal variables in a set order, which involves a separate process. These types of models are known as SARIMA.
